Etiquetas HTML

HTML significa "HyperText Markup Language". Es un lenguaje de marcado utilizado para crear páginas web. Las etiquetas HTML se utilizan para definir el contenido y la estructura de una página web.

Etiqueta de Encabezado

Etiqueta de Subencabezado

Las etiquetas de encabezado se utilizan para definir el título y los sub-títulos de una página web. Hay seis niveles de encabezado, desde h1 hasta h6, donde h1 es el más importante y h6 es el menos importante.

La etiqueta hr se utiliza para crear una línea horizontal en la página web.

La etiqueta ul se utiliza para crear una lista desordenada en la página web. Cada elemento de la lista se define con la etiqueta li.

La etiqueta ol se utiliza para crear una lista ordenada en la página web. Cada elemento de la lista se define con la etiqueta li.

Enlace

La etiqueta a se utiliza para crear un enlace en la página web. El atributo href define la URL a la que se debe dirigir el enlace.

La etiqueta img se utiliza para insertar una imagen en la página web. El atributo src define la URL de la imagen, mientras que el atributo alt se utiliza para describir la imagen.

La etiqueta table se utiliza para crear una tabla en la página web. Las filas se definen con la etiqueta tr, mientras que las celdas se definen con la etiqueta td. Las celdas de encabezado se definen con la etiqueta th.
